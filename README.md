# Plantas - Mi Primera Página

Página web sobre plantas naturales, desarrollada con HTML, CSS y Bootstrap 5.

## Estructura del Proyecto

```
mi-primera_pagina/
├── index.html
├── css/
│   └── styles.css
└── README.md
```

## Características

- Diseño responsivo con Bootstrap
- Carrusel de imágenes
- Cards interactivas con información de plantas
- Formulario de contacto

## Cómo Ejecutar

1. Abrir `index.html` en un navegador web

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5.3.3
